# Whole installed programs (managed by this dotfiles)

This is the list of all programs managed or installed by this dotfiles.
Some of these list are auto-generated by `containers/dependencies/gen.py`.

## base development deps

minimal required softwares that is needed to `apply` this dotfiles.
These softwares are required to run `chezmoi apply` successfully.
And softwares like `mise` are excluded, because it isn't required to apply.
(But it is needed for running zsh or other programs)

Now it is written only for `Manjaro` or `Arch` OS environment.

- base-devel (aur)
- git
- chezmoi
- rbw

## Container management deps

Softwares needed to start podman container under `containers/`

- make
- git
- podman

## Config managed softwares

Softwares that configs are managed in this dotfiles

- chezmoi
  - $HOME/.chezmoidata.yaml (`.chezmoidata.yaml`)
  - $HOME/.chezmoiignore (`.chezmoiignore`)
- rbw
  - $HOME/.chezmoidata/rbw.yaml (`dot_config/rbw/config.toml`)
    - Used for injecting secrets into chezmoi templates
  
- zsh
  - $HOME/.zshenv (`dot_zshenv`)
  - $HOME/.zshrc (`dot_zshrc`)
- mise
  - $HOME/.config/mise/config.toml (`dot_config/mise/config.toml`)
- direnv
  - $HOME/.config/direnv/direnvrc (`dot_config/direnv/direnvrc`)
- sheldon
  - $HOME/.config/sheldon/plugins.toml (`dot_config/sheldon/plugins.toml`)
- gnupg
  - $HOME/.local/share/gnupg/ (`dot_local/share/gnupg/`)
